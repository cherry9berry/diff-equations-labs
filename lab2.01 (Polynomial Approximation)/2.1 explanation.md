# Lab 2.1: Polynomial Approximation of ODE Solutions

## Work Description
This laboratory work explores the method of approximating dedicated Cauchy problem solutions using Taylor polynomials. When an analytical (exact) solution to a differential equation is difficult or impossible to find in elementary functions, we can represent the unknown function as a power series. By calculating the derivatives at the initial point, we construct a polynomial that closely follows the true solution curve in the vicinity of that point. 

The primary goal is to compare how different degrees of Taylor polynomials (4th and 5th) perform against high-precision numerical integration methods like Runge-Kutta (RK4/RK45).

## Mathematical Model
We consider the following first-order nonlinear differential equation:
$$y' = x^2 + y^3, \quad y(1) = 1$$

To find the Taylor polynomial coefficients, we calculate higher-order derivatives at $x_0 = 1$ using the original equation:
1. $y'(1) = 1^2 + 1^3 = 2$
2. $y'' = 2x + 3y^2 y'$ $\implies y''(1) = 2(1) + 3(1)^2(2) = 8$
3. $y''' = 2 + 6yy'^2 + 3y^2 y'' \implies y'''(1) = 2 + 6(1)(2^2) + 3(1)^2(8) = 50$
... and so on.

The Taylor polynomial of degree $n$ is given by:
$$P_n(x) = \sum_{k=0}^{n} \frac{y^{(k)}(x_0)}{k!} (x - x_0)^k$$

## Implementation & Results
In the notebook, polynomials of 4th and 5th degrees were implemented:
- **Poly-4**: $y(x) \approx 1 + 2(x-1) + 4(x-1)^2 + \frac{25}{3}(x-1)^3 + \frac{81}{4}(x-1)^4$
- **Poly-5**: Adds the term $\frac{1027}{20}(x-1)^5$

The results show that while polynomials are extremely accurate very close to $x=1$, they begin to diverge rapidly as we move further away (e.g., at $x=1.3$), especially for nonlinear equations where the solution might grow very fast. In contrast, numerical methods like **RK4** maintain high stability and accuracy over a wider range.
