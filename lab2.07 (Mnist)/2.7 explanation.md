# Lab 2.7: MNIST Digit Recognition

## Work Description
This laboratory work focuses on implementing a Fully Connected Neural Network (also known as a Multi-Layer Perceptron) from scratch using only NumPy. The goal is to classify handwritten digits from the famous **MNIST dataset**.

The task involves:
1.  Handling and preprocessing high-dimensional image data ($28 \times 28$ pixels).
2.  Implementing the core logic of neural networks: forward propagation and backpropagation.
3.  Optimizing the model using mini-batch gradient descent to achieve high classification accuracy.

## Mathematical Model
The network uses the following architecture:
-   **Input Layer**: 784 neurons (each representing one pixel).
-   **Hidden Layer**: 128 neurons with **Sigmoid** activation.
-   **Output Layer**: 10 neurons (one for each digit 0-9) with **Softmax** activation to produce a probability distribution.

### Optimization and Learning
-   **Loss Function**: Categorical Cross-Entropy (implied by the Softmax/Gradient derivation).
-   **Backpropagation**: Calculates gradients of the loss with respect to weights ($W$) and biases ($b$) using the chain rule.
-   **Weight Update**: $W = W - \eta \cdot \nabla W$, where $\eta$ is the learning rate.

## Implementation & Results
-   **Epochs and Batches**: The model trains for 15 epochs with a batch size of 128.
-   **Accuracy**: The implementation reaches approximately **93-94% accuracy** on both training and test sets, demonstrating good generalization.
-   **Visualization**: The notebook includes a chart showing convergence (loss reduction / accuracy increase) and a sample of correctly/incorrectly classified digits.
