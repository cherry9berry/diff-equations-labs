# Lab 2.4: Method of Matched Asymptotic Expansions

## Work Description
This laboratory work investigates boundary value problems (BVP) for second-order differential equations where a small parameter $\epsilon$ multiplies the highest derivative. This situation typically results in a **boundary layer** — a narrow region where the solution changes rapidly to satisfy boundary conditions.

The target equation is:
$$\epsilon y'' + (2x+1)y' + 2y = 0, \quad x \in [0, 1]$$
with boundary conditions:
$$y(0) = A, \quad y(1) = B$$

The objectives are:
1. Construct the **outer solution** (valid away from the boundary layer).
2. Construct the **inner solution** (valid inside the boundary layer at $x=0$).
3. Form a **composite solution** using the matching principle.
4. Solve the problem numerically using the **finite difference method** (Thomas algorithm/tridiagonal matrix algorithm) and compare with the analytical expansion.

## Mathematical Model
For $\epsilon \to 0$, the "degenerate" equation is $(2x+1)y' + 2y = 0$.
- **Outer Solution ($y_{out}$)**: Solving the first-order equation and satisfying $y(1)=B$ gives $y_{out}(x) = \frac{3B}{2x+1}$.
- **Inner Solution ($y_{in}$)**: Introducing a stretched variable $\xi = x/\epsilon$ and solving the resulting boundary layer equation gives an exponential term.
- **Composite Solution ($y_{comp}$)**: By matching the solutions and removing the overlap, we get:
$$y_{comp}(x) \approx \frac{3B}{2x+1} + (A - 3B)e^{-x/\epsilon}$$

## Implementation & Results
- **Numerical Methods**:
    - **Progonka (Thomas Algorithm)**: A custom finite difference scheme was implemented.
    - **SciPy solve_bvp**: Used as a high-precision reference.
- **Observations**: The boundary layer near $x=0$ is clearly visible in the plots. As $\epsilon$ decreases, the layer becomes sharper. The composite analytical solution provides an excellent approximation to the numerical results, with accuracy improving as $\epsilon$ becomes smaller.
