# Lab 2.5: Linear Regression, OLS, and Gradient Descent

## Work Description
This laboratory work explores the fundamental methods of machine learning and statistics: **Simple Linear Regression**. The goal is to build a model that predicts a dependent variable $y$ based on an independent variable $x$ using a linear relationship:
$$y_i = w x_i + b + \varepsilon_i$$
We use two different approaches to find the optimal parameters $w$ (weight/slope) and $b$ (bias/intercept):
1. **Ordinary Least Squares (OLS)**: An analytical method that finds the exact solution by minimizing the sum of squared residuals.
2. **Gradient Descent**: An iterative optimization algorithm that approaches the minimum of the error function by following the negative gradient.

Additionally, the work includes statistical validation of the model using **Fisher's Criterion (F-test)** to determine if the regression is statistically significant.

## Mathematical Model

### 1. Ordinary Least Squares (OLS)
OLS minimizes the **Residual Sum of Squares (RSS)**:
$$Q(w,b) = \sum_{i=1}^{n} (y_i - (w x_i + b))^2 \to \min$$
Solving the system of normal equations leads to:
$$\hat w = \frac{n S_{xy} - S_x S_y}{n S_{xx} - S_x^2}, \quad \hat b = \bar y - \hat w \bar x$$

### 2. Gradient Descent
Gradient descent updates parameters iteratively:
$$w \leftarrow w - \alpha \frac{\partial Q}{\partial w}, \quad b \leftarrow b - \alpha \frac{\partial Q}{\partial b}$$
Where $\alpha$ is the learning rate. For linear regression, the loss surface is convex, ensuring convergence to the OLS solution.

### 3. Fisher's Criterion (F-test)
Used to test the null hypothesis $H_0: w = 0$ (the model is not better than a simple mean).
$$F = \frac{SSR / 1}{SSE / (n - 2)}$$
If the calculated $F$ is greater than the critical value from the F-distribution table, the model is considered statistically significant.

## Implementation & Results
- **Data Normalization**: Standardizing features (Z-score normalization) significantly speeds up Gradient Descent convergence.
- **Comparison**: Both OLS and Gradient Descent yield nearly identical results for the regression line.
- **Statistical Significance**: F-statistic and $R^2$ (coefficient of determination) are calculated to measure how well the model explains the variance in the data.
